Git Steps: 

1. Create new folder
2. Right click folder n Open Git Bash
3. Inside folder, create new file test.txt
4. In Git Bash:
	- git init
	- git status
	- git add file-name (Stag)..Multiple files: git add .
	- git status
	- git commit -m "Comments"
	- git branch -M main
	- git remote add origin (link in Github)
	- git push -u origin main

5. If changes made from Github:
	- Edit the file and commit
	- git pull origin main  //This will make changes in the local

6. To create New Branch:
	- git branch branch_name
	-  git checkout testbranch //Git bash is pointing to main
